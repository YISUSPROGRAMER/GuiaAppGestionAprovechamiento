# GuiaApp Gestión de Aprovechamiento

Aplicación Web Progresiva (PWA) para la gestión de recolección de residuos y aprovechamiento.

## Despliegue en GitHub Pages

Esta aplicación se despliega automáticamente en GitHub Pages usando GitHub Actions.

**Estado del Despliegue:**
[![Deploy to GitHub Pages](https://github.com/YISUSPROGRAMER/GuiaAppGestionAprovechamiento/actions/workflows/deploy.yml/badge.svg)](https://github.com/YISUSPROGRAMER/GuiaAppGestionAprovechamiento/actions/workflows/deploy.yml)

**URL de la Aplicación:**
[https://YISUSPROGRAMER.github.io/GuiaAppGestionAprovechamiento/](https://YISUSPROGRAMER.github.io/GuiaAppGestionAprovechamiento/)

## Configuración Local

1.  Instalar dependencias: `npm install`
2.  Correr en desarrollo: `npm run dev`
3.  Construir para producción: `npm run build`
